# Due Date Calculator

Implemented by [rpi1337](https://twitter.com/rpi1337)


## Prerequisites

 - NodeJS 10.0 and above
 - NPM 6.0 and above


## Build

```
npm install
npm run build
```

## Tests

```
npm run test
 ```
## Runing

Once running you have to feed submission date as Unix timestamp and the turnaround time in hours 

```
npm start 1571493720000 10
```
